octopus - distributed version control system

The DVCS system, dubbed, octopus, will help you backup and track your files. It will also help share your changes and projects with others. This system is centralized and simple - everyone synchs and checks into the main repository.

The main feutures include:

  • clone
  • push and pull
  • commit
  • statux
  • branch
  • checkout

From the implementation side, the system was devided into 5 modules: UI, Workspace, Repo, Revlog and Push and Pull. Check out this Github page for the source code and usage instructions.
